# ✨05-chainlit: Minimal Echo Bot💬

*A no-frills Chainlit starter to get your AI chat game rolling. Just type, hit enter, and it talks back—literally. Great for prototyping or learning how Chainlit works.*

🔧 What It Does
User: “Hello?”
Bot: “Received: Hello?”

That’s it. For now.
